About Me

Hey I'm Roisin, a 19 year old Design student at Victoria University of Wellington, currently majoring in Media Design and Information Systems. My passion within my work revolves around creating innovative projects based on the web that educate, inspire or entertain others.


Technology is becoming an essential part of our lives, creating so many opportunities for benefits within our society. I feel this is an area I would love to delve into in my future work, combining my skills of programming and design.


This website exists to showcase my design work and present future ideas as I try to find which creative direction I want to head in.

Recent Projects

Spot my Mate

Spot my Mate is an interactive mapping project, designed to get participants to use technology to increase communication in real life.

This project involves a location saving website which can be viewed at www.spotmymate.com, Arduino microcontrollers, Code written in Processing and an RGB LED matrix

Visit the website here

You Are Here

You are Here tracks location data from a users daily journey and outputs it as a beautiful unique piece of art

This project uses a GPS module connected to an Arduino to creating a portable GPS tracking device that outputs its data to a journey visualiser program. Through Java this program creates a visual representation of this journey.
